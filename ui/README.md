# angularjs [![build](https://travis-ci.org/daggerok/angularjs.svg?branch=master)](https://travis-ci.org/daggerok/angularjs)

bundle angularjs using webpack2

```bash
npm i
npm t
npm start
npm run lint
npm run build
npm run serve
npm run ghpages
npm run ghserve
fish bin/gh-pages.fish
```


update npm versions

```bash
npm i -g npm-check-updates
ncu -u
```
